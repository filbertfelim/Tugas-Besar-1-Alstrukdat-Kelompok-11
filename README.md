#### Cara mengcompile file program :

- ##### Langkah 1 : ( Mengubah current directory ke bin )<br />
      cd bin

- ##### Langkah 2 : ( Mengcompile file .c )<br />
      make driver

- ##### Langkah 3 : ( Menjalankan program )<br />
      ./driver


##### Jika ingin menghapus executable program :<br />
    make clean
